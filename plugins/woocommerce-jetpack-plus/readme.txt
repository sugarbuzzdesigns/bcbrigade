== Installation ==

1. Upload the entire 'woocommerce-jetpack-plus' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce > Settings > Booster.

== Frequently Asked Questions ==

= Should I leave Booster for WooCommerce plugin enabled and activated when I install Booster Plus? =

Yes, Booster for WooCommerce plugin MUST be installed and activated when you install Booster Plus.
