Theme Switcher Reloaded is an updated and much improved version of the original Theme Switcher. Comes with a widget and can also switch themes via URL.

== Description ==

Theme Switcher Reloaded is sponsored by Themebot for use with the demos in the [WordPress Themes](http://themebot.com/wordpress-themes "WordPress Themes") section on themebot.com. Coding was done by kingler of [72pines](http://code.72pines.org/ "72pines"). This is an updated version of the venerable themeswitcher by Ryan Boren and Theme Switcher Widget by Jared Bangs. Theme Switcher Reloaded adds new features and enhancements.

== Installation ==

1. Upload `theme-switcher-reloaded` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Activate the widget and select option for use in the sidebar.
4. If you only want to use the URL loading functionality, you may deactivate the widget after saving options.
